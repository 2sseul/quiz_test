# 📚 AI 퀴즈 변환기

> GPT · Gemini가 만들어준 퀴즈 텍스트를 붙여넣으면 바로 풀 수 있는 퀴즈 UI로 변환해주는 웹앱

**[→ 바로 사용하기](https://2sseul.github.io/quiz-app/)**

회원가입 없이 브라우저에서 바로 사용할 수 있어요.  
오답노트는 내 브라우저에 자동으로 저장됩니다.

---

## ✨ 기능

**프롬프트 생성기**  
원하는 문제 유형(객관식 / 단답형 / 서술형)과 문제 수를 선택하면 AI에게 보낼 프롬프트를 자동으로 만들어줘요. 복사해서 GPT나 Gemini에 붙여넣고 강의자료를 첨부하면 끝.

**퀴즈 변환**  
AI가 만들어준 텍스트를 그대로 붙여넣으면 풀 수 있는 퀴즈 카드로 바꿔줘요. 객관식·단답형·서술형 모두 지원하고, 정답과 해설도 자동으로 파싱됩니다.

**두 가지 풀기 모드**  
- **시험지 모드** — 전체 문제를 쭉 풀고 마지막에 한꺼번에 채점
- **한 문제씩 모드** — 문제마다 바로 채점하고 해설 확인

**자동 채점**  
- 객관식 → 선택 즉시 정답 표시
- 단답형 → 키워드 매칭으로 자동 채점
- 서술형 → 모범답안 제시

**오답노트**  
틀린 문제가 자동으로 저장됩니다. 나중에 오답만 골라서 랜덤 순서로 다시 풀 수 있어요.

**세트 저장 · 내보내기**  
퀴즈 세트를 이름 붙여 저장하고 언제든 불러올 수 있어요. 노션에 바로 붙여넣을 수 있는 마크다운 형식으로 내보내기도 가능합니다.

---

## 🚀 사용법

### 1단계 — 프롬프트 복사
앱 상단에서 문제 유형과 수를 선택하고 **프롬프트 복사하기** 버튼 클릭

### 2단계 — AI에게 요청
GPT · Gemini 채팅창에 붙여넣고, 강의자료 파일을 첨부해서 전송

### 3단계 — 결과 붙여넣기
AI가 출력한 퀴즈 텍스트를 **입력** 탭에 붙여넣고 **변환하기** 클릭

### 4단계 — 퀴즈 풀기
시험지 모드 또는 한 문제씩 모드로 풀고, 틀린 문제는 오답노트에서 다시 풀기

## 📄 지원 포맷

AI가 아래 형식으로 출력하면 자동으로 파싱돼요.  
앱의 프롬프트 복사 기능을 쓰면 이 형식대로 자동으로 요청됩니다.

```
Q1. 문제 내용
① 보기 1
② 보기 2
③ 보기 3
④ 보기 4
정답: ②
해설: 정답 이유

Q2. [단답형] 문제 내용
정답: 키워드
해설: 이유

Q3. [서술형] 문제 내용
모범답안: 모범 답안
해설: 핵심 포인트
```
